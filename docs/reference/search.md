# Search

This documentation set is designed for navigation-first browsing, with GitHub and IDE search filling in the gaps for full-text queries.

## Prerequisites

- None

## How to find things

Start with the indexes:

- `README.md` for the top-level entry point
- `docs/navigation.md` for a map of the tree
- Category indexes:
  - `docs/getting-started/index.md`
  - `docs/features/index.md`
  - `docs/operations/index.md`
  - `docs/contributing/index.md`
  - `docs/architecture/index.md`
  - `docs/reference/index.md`

Then use search when you know what you want:

- GitHub: use the repository search bar (supports file-path filters like `path:docs`)
- IDE: search in files for settings, env var names, and script names

Useful keywords:

- `docker compose` (runtime commands)
- `WORKSPACE_PATH` (mount configuration)
- `LOCAL_UID` / `LOCAL_GID` (permissions)
- `AGENT_` (agent/assistant settings)
- `MCP` (MCP integration)
- `chezmoi` / `age` (secrets management)
